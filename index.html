一部《寻梦环游记》湿润了无数人的眼眶，而动画中五彩缤纷的墨西哥小城和亡灵小镇，也让人们心生遐想。

作为原型的瓜纳华托，每一条街道，每一片砖墙都是彩色的，更是被国家地理和BBC评为一生最值得去的地方。

加勒比的海面平静清澈，因其深浅、海底生物情况和阳光照射等原因，会呈现出白色、天蓝、深蓝、黑色等多种颜色，沿海的小岛拥有无比浪漫奢华的度假体验。

在玛雅语中，坎昆意为“挂在彩虹一端的瓦罐”，被认为是欢乐和幸福的象征。坎昆有世界公认的十大最美海滩之一，在洁白细腻的沙滩上享受加勒比的阳光，是人们心中对悠闲度假的最高享受。

去牙买加品尝最地道正宗的贵族级蓝山咖啡，观赏加勒比海唯一的临海瀑布，如婚礼蛋糕般的梦幻多姿。

深藏在雨林之中的玛雅城市遗址，自从被发掘以来一直吸引着科学家们探索的目光。

奇琴伊察的的数百座古玛雅文化和托尔特克文化的遗址，向人们展示了典雅精致的建筑风格和神秘莫测的浮雕图案。依玛雅历法而建的库库尔坎金字塔更是显示出了古代玛雅人高超先进的技术和失落在历史长河中的神话智慧。

被称为“中美洲的花园”的哥斯达黎加，有着秀美异常的自然风光，境内有9座活火山依然活跃着喷出沸腾的蒸汽和岩浆。热带雨林和奇珍动物，让这儿仿若动画中的动物王国。

阿雷纳火山国家公园内，拥有世界上最活跃的火山之一——阿雷纳火山，火山附近有数条热气蒸腾的温泉河，游人们可以来此修养泡温泉，感受与众不同的温泉体验。

在Casco Viejo老城区漫步，一边是焕然一新的西班牙殖民建筑，一边是遗留下来的残垣断壁，可以想象出17世纪西班牙水军统治时期的辉煌景象。

巴拿马城的西部，是著名的南北美洲分界——巴拿马运河，这条运行了100多年的运河至今仍发挥着巨大的作用，河道口的古堡经历了四百年的风雨依然完好的屹立在那儿，最南段的炮台城墙直面大海，能看到美洲大桥。

在世外桃源般的岛国巴哈马，隐藏着一片娟秀绮丽的粉红沙滩。在这片被美国《新闻周刊》评为“世界上最性感的沙滩”的度假天堂，湛蓝的天空，澄澈的海水和淡粉色的细沙完美的融成一片画境。

除了仙境般的风光，巴哈马还有萌化人心的猪岛，在这里生活的可爱野生小猪和人类和谐共处。每当有游客前来，活泼的小猪们还会热情的游泳前来，讨要食物。

激情洋溢的哈瓦那素有加勒比海的明珠之称。旧城区里古老的教堂，百年城堡，记录了西班牙殖民文化的博物馆，和现代化的新城比邻而居，历史和现实交相辉映。

而当经典复古的老爷车和醇厚而内敛的鸡尾酒在哈瓦那相遇，会碰撞出什么样的火花呢？乘坐古朴帅气的老爷车，游览哈瓦那观光，任微风拂过发梢，体验城区古老精美的建筑和现代化的气息。

基韦斯特是美国本土最南端的一个珊瑚岛。

连接美国大陆和基韦斯特的跨海大桥长达一百多英里，被称为“世界第八大奇迹”，是世界上最美的公路。很多人来此甚至就是为了体会这条海中公路沿途的无限风光，以及和古巴遥遥相望的美国最南端的纪念塔。

海明威曾居住在这座宁静的小岛，并完成了《乞力马扎罗的雪》等著作。岛上至今仍居住着六趾猫的后代，每一处都散发着慵懒的假日气息。这儿色彩斑斓的建筑和透彻的海水与海滩，让人们仿佛置身世外桃源。

    圣何塞 → 阿雷纳

    早餐后市区观光，午餐后驱车前往哥斯达黎加最有名的景点之一——阿雷纳火山。途中会经过被誉为艺术家摇篮的城市隆儿奇，路上还可参观铁皮教堂；下午抵达阿雷纳火山区。

    随后由导游带领，直接入住酒店，享受温泉。

  • Day9

    阿雷纳 → 曼纽尔安东尼奥国家公园

    早餐后，参观阿雷纳火山国家公园。之后驱车前往安东尼奥国家公园。

  • Day10

    曼纽尔安东尼奥国家公园 → 圣何塞

    曼纽尔安东尼奥国家公园位于太平洋沿岸，是哥斯达黎加第二大旅游目的地。除了可以看到各种陆地动物和鸟类，还可以享受沙滩和海洋，欣赏海洋生物。 参观后返回圣何塞，入住酒店休息。

  • Day11

    圣何塞 → 巴拿马城

    早餐后送机前往巴拿马首都巴拿马城，抵达后接机。游览Casco Viejo老城区。

    巴拿马城 → 金斯敦

    金斯敦

    前往牙买加岛上最长的山脉——蓝山,游客可在海拔3100英尺的草莓山庄观赏蓝山咖啡的制作过程。

    金斯敦 → 奥乔里奥斯

    奥乔里奥斯 → 金斯敦

    金斯敦 → 哈瓦那

    哈瓦那

    早餐后全天哈瓦那城市游览。特别安排老爷车城市观光之旅。上午新城区游览，下午前往老城区。傍晚在莫罗城堡参观换岗仪式。

    哈瓦那 → 拿骚
