乘马车巡游西西里岛首府巴勒莫，它被歌德称为“世界上最优美的海岬”，高大的教堂和随处可见的壁画散发着西西里浓郁的艺术气息。

在卡塔尼亚看一场西西里传统木偶剧，专业工匠用繁复的雕刻与着色制作出精美木偶，演绎着宏伟史诗和英雄传奇，是独属西西里之世界文化遗产的独特魅力。

前往有着许多珍稀动植物的美丽岛，这是一片被世界自然基金会划为自然保护区的唯美之地。标志性的心形沙滩，有着西西里独特的浪漫风情。

游览阿玛尔菲海岸，这个“绝美而典型的地中海风光”的拥有者，1997年被联合国列入世界遗产名录，1999年被美国《国家地理》评选为一生必去。

乘船前往卡布里岛的诗情画意，从古罗马时代起这里就是著名的疗养胜地。奥古斯都大帝曾用大4倍的海岛换了这片碧海仙境，并且一住就是10年。

在船夫的带领下，乘舟游览位列世界七大奇景之一的蓝洞，池水呈现出蓝宝石般的色彩映射在两侧石壁之上，魔幻而静谧。

游览阿玛菲海岸最美小镇波西塔诺，如画风景让它成为社会名流的梦幻后花园。100多年以前的毕加索就曾对它流连忘返。

前往世界文化遗产庞贝古城，它在2000年前倍火山喷发淹没，古罗马的生活日常也在那一刻被永久定格。这里一经发掘，便震动全世界。

漫步阿格里真托 “神殿之谷”，这片神奇的土地从公元前6世界跨越至今，拥有着全世界保存最完好的古希腊神庙群，并于1997年被列入《世界遗产名录》。

走进锡拉库萨古城，它的美丽常被历代文人写入诗歌，它是阿基米德的故乡，也是《西西里的美丽传说》的知名取景地。

乘越野车四驱车探险欧洲最大活火山埃特纳，千百年的喷发在西西里造就了一片瑰丽奇绝的景观，位列西西里众多世界自然遗产之一。

游览参观位列世界文化遗产蒙雷阿莱大教堂，它让拜占庭镶嵌艺术、阿拉伯建筑风格以及诺曼式样和谐相融，其恢弘精美，在全世界绝无仅有。

同样位列世界遗产名录的卡萨尔古罗马别墅，其建造者被普遍认为是罗马帝国贵族元老甚至是皇家成员，其大面积的马赛克装饰巅峰造极，向我们完美展示了当时的生活和远古神话。

前往希腊圆形露天剧场，建在崖壁上，悬浮于海天之间，大剧场的古迹戏台和观众座位仍被完整保留，是世界上最具特色的希腊式剧院。

  • Day1

    全国各地 → 巴勒莫

    搭乘国际航班前往迪拜，抵达后接机。

  • Day2

    巴勒莫 → 切法卢 → 巴勒莫

    1. 上午乘马车游览巴勒莫古城，参观巴勒莫大教堂、诺曼王宫、四角场
    2. 下午前往海边小镇切法卢，游览切法卢老城，切法卢大教堂

  • Day3

    巴勒莫 → 蒙雷阿莱 → 阿格里真托

    1. 上午前往巴勒莫市郊的蒙雷阿莱，参观蒙雷阿莱大教堂
    2. 前往阿格里真托，游览土耳其阶梯，漫步神殿之谷

  • Day4

    阿格里真托 → 卡萨尔古罗马别墅 → 锡拉库萨

    1. 上午前往游览卡萨尔的古罗马别墅
    2. 下午前往阿基米德的故乡锡拉库萨
    3. 参观锡拉库萨古城、奥尔迪娜岛

  • Day5

    锡拉库萨 → 陶尔米纳

    1. 前往陶尔米纳，参观希腊露天圆形剧场、陶尔米纳大教堂，四月九日广场
    2. 游览心形海滩的美丽岛

  • Day6

    陶尔米纳 → 埃特纳火山 → 卡塔尼亚 → 陶尔米纳

    1. 上午越野车四驱车探险埃特纳活火山
    2. 下午前往卡塔尼亚自由观光，欣赏西西里木偶剧

  • Day7

    陶尔米纳 → 卡塔尼亚 → 那不勒斯 → 索伦托

    1. 上午前往机场，搭乘航班前往那不勒斯。
    2. 下午前往阿玛菲海岸索伦托，独享无边泳池迷人海景

  • Day8

    阿玛菲海岸 → 卡布里岛 → 阿玛菲海岸

    1. 上午乘船前往卡布里岛，搭缆车抵达山顶
    2. 探秘世界七大奇景之一的蓝洞
    3. 下午逛阿玛菲海岸最美的小镇波西塔诺

  • Day9

    索伦托 → 庞贝 → 罗马

    1. 上午游览参观庞贝古城
    2. 下午前往罗马，抵达后游览西班牙广场，许愿池

  • Day10

    罗马 → 国内

    1. 罗马城市参观或自由购物
    2. 傍晚前往机场，搭乘航班返回国内，结束愉快旅程
